# 项目简介

## 核心功能
- **在住管理**: 管理社区养老院内的居民信息，包括基本信息、入住时间等。
- **护理计划**: 制定个性化的护理计划，根据居民需求和健康状况进行定制。
- **护理等级**: 设定护理等级，根据等级分配相应的护理资源和服务。
- **护理项目**: 管理具体的护理项目，如日常护理、康复训练等。
- **AI智能评测**: 利用AI大模型进行智能评测，提供更为精准的健康评估。
- **体检报告**: 自动生成体检报告，提供详细的健康数据。

## 适用场景
- 社区养老院
- 养老服务机构
- 健康管理机构

## 技术特点
- **开发环境**: IDEA、JDK17、Maven3.8.8、Node.js 21
- **技术栈**: Java、SpringBoot、MyBatis-Plus、Redis、SpringSecurity权限管理、Vue3
- **数据库**: MySQL
- **前后端分离**: 采用前后端分离架构，提高开发效率和系统可维护性。
- **可视化界面统计图**: 提供直观的统计图表，便于管理人员快速掌握养老院运营状况。
